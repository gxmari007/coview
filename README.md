# CoView

A Vue.js 2.0 ui components for web

## Install

```` bash
# install
npm install coview --save
````

## Usage

````javascript
import Vue from 'vue';
import CoView from 'coview';

Vue.use(CoView);
````

or

````javascript
import Vue from 'vue';
import { CoButton, CoInput } from 'coview';

Vue.component(CoButton.name, CoButton);
Vue.component(CoInput.name, CoInput);
````
